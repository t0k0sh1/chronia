    Function compare

    • compare(date1: Date, date2: Date, order?: "ASC" | "DESC"): number

      Compare two Date objects chronologically.

      Parameters

      • date1: Date

        The first Date object to compare

      • date2: Date

        The second Date object to compare

      • order: "ASC" | "DESC" = "ASC"

        Optional sort order: "ASC" for ascending (default) or "DESC" for descending

      Returns number

      -1 if date1 < date2, 1 if date1 > date2, 0 if equal (adjusted for order)

      Throws

      When arguments are not valid Date objects or order is invalid

      Example

      compare(new Date('2024-01-01'), new Date('2024-01-02')); // -1 (ascending)
      compare(new Date('2024-01-01'), new Date('2024-01-02'), 'DESC'); // 1 (descending)
      compare(new Date('2024-01-01'), new Date('2024-01-01')); // 0 (equal)

      Example

      // Sort dates in ascending order
      dates.sort(compare);

      Example

      // Sort dates in descending order
      dates.sort((a, b) => compare(a, b, 'DESC'));
